# Feature Inventory

## Core Timer Features

### Multi-Timer Management
- **Unlimited Timers** (Premium) / **3 Timer Limit** (Free)
- **Simultaneous Running**: Multiple timers can run concurrently
- **Timer Categories**: Organize timers by project or type
- **Priority Levels**: High (3), Medium (2), Low (1) priority system
- **Deadlines**: Set and track timer-specific deadlines
- **Drag & Drop Reordering**: Custom timer organization
- **Timer Persistence**: Automatic save/restore across sessions

### Timer Controls
- **Start/Stop/Reset**: Basic timer operations
- **Real-time Updates**: Live elapsed time display
- **Session Tracking**: Individual timer sessions stored
- **Browser Events**: Handle tab close/refresh gracefully
- **Automatic Recovery**: Restore running timers on page load
